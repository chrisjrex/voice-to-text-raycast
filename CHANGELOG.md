# Voice-To-Text Changelog

## [Initial Version] - 2026-02-15

- Speech-to-text transcription using Whisper and Parakeet models via Apple MLX
- Text-to-speech with Kokoro, Piper, and macOS system voices
- AI-powered post-processing with configurable processors
- Custom dictionary for protected terms
- Transcription history
- TTS menu bar with Kokoro server management
- Per-command settings for transcription, model management, and TTS
